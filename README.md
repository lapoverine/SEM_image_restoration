# SEM Image Restoration
Совместное шумоподавление и сверхразрешение для электронной микроскопии. При сканировании данных ЭМ возникают помехи, и в них практически отсутствует парная достоверность, поэтому преобладают методы самоконтроля (Noise2Noise, Noise2Void, Noise2Self) и методы с нулевым разрешением.
